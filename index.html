2. FUNDAMENTALS

HTML

hypertext markup language is used to structure content for web browsers.

DOM

the document object model refers to the hierarchical structure of html. each bracketed tag is an element, and we refer to elements' relative relationship to each other in human terms. parent, child, sibling, ancestor, and descendant. in this html code, the body element is a parent element to all of its children elements. all elements on the page are descendants of html. we browsers parse the DOM in order to make sense of a page's content.

CSS

cascading style sheets are used to make visual presentation of HTML.

JavaScript

javascript is a dynamic scripting language that can instruct the browser to make changes to a page after it has already loaded. scripts can be included directly in HTML, between two script tags.

Developer Tools

while 'view source' shows you the original HTML source of the page, the web inspector shows the current state of the DOM. this is useful because your code will modify DOM elements dynamically. in the web inspector, you can watch elements as they change. you'll also use the JAVASCRIPT console for debugging. see more on debugging HTML, CSS, and JAVASCRIPT with the web inspector and console. SVG

D3 is at its best when rendering visuals as SCALABLE VECTOR GRAPHICS (SVG). SVG is a text-based image format. this means that SVG images are specified by simply writing mark-up. SVG is awesome.
